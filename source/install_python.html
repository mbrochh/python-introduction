Install Python on OSX

It is installed. If you ever want a "fresh" install, you can do this (but really don't need to for this tutorial):
  1. Install Homebrew: http://brew.sh
  2. Install Python 
    brew install python
  3. Where is it? 
    brew list python
# You don't really need to know this
  4. Check Installation: 
    python --version

Install Python on Ubuntu

It is already installed, but you need some extras
  1. Update software 
    sudo apt-get update && sudo apt-get upgrade
  2. Install Python dependencies 
    sudo apt-get install python-dev
  3. Where is it? 
    /usr/bin/python
/usr/local/lib/python2.7
 # You don't really need to know this
