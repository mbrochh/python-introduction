Install Virtualenv on OSX & Ubuntu (I)

Virtualenv helps to keep each environment for each project clearly separated

  1. Install pip 
    sudo easy_install pip
  2. Install virtualenv 
    sudo pip install virtualenv
  3. Install virtualenvwrapper 
    sudo pip install virtualenvwrapper

Install Virtualenv on OSX & Ubuntu (II)

  1. Open the file `.bashrc` or `.bash_profile` in an editor 
    # On OSX:
touch ~/.bash_profile && open ~/.bash_profile
# On Ubuntu:
gedit ~/.bashrc
  2. Add to the end of the file: 
    export WORKON_HOME=~/Envs
export PIP_VIRTUALENV_BASE=~/Envs
# On OSX
source /usr/local/bin/virtualenvwrapper.sh
# On Ubuntu
source /usr/share/virtualenvwrapper/virtualenvwrapper.sh
