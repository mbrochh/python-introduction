Install Python on OSX

Should already be installed. If you want a "fresh" install, you can do this:
  1. Install Homebrew: http://brew.sh
  2. Install Python 
    brew install python
  3. Where is it? 
    brew list python
# You don't really need to know this
  4. Check Installation: 
    python --version

Install Python on Ubuntu

Should already be installed, but we need some extras
  1. Update software 
    sudo apt-get update && sudo apt-get upgrade
  2. Install Python dependencies 
    sudo apt-get install python-dev
  3. Where is it? 
    /usr/bin/python
/usr/local/lib/python2.7
 # You don't really need to know this

Install Virtualenv on OSX & Ubuntu (I)

  1. Install pip 
    sudo easy_install pip
  2. Install virtualenv 
    sudo pip install virtualenv
  3. Install virtualenvwrapper 
    sudo pip install virtualenvwrapper

Install Virtualenv on OSX & Ubuntu (II)

  1. Open the file `.bashrc` in an editor 
    # On OSX:
open ~/.bashrc
# On Ubuntu:
gedit ~/.bashrc
  2. Add to the end of the file: 
    export WORKON_HOME=~/Envs
export PIP_VIRTUALENV_BASE=~/Envs
# On OSX
source /usr/local/bin/virtualenvwrapper.sh
# On Ubuntu
source /usr/share/virtualenvwrapper/virtualenvwrapper.sh
